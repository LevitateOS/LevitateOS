# Security Policy

## Supported Versions

We currently support the latest version of LevitateOS on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within LevitateOS, please do not report it through public issues or pull requests. Instead, please follow these steps:

1.  **Initial Contact**: Send an email to the project maintainers (link to be provided or via a secure channel).
2.  **Information Gathering**: Provide a detailed description of the vulnerability, including steps to reproduce it and a potential impact assessment.
3.  **Coordinated Disclosure**: We will acknowledge receipt of your report and work with you to understand and resolve the issue. We ask that you give us reasonable time to fix the vulnerability before disclosing it publicly.

Thank you for helping keep LevitateOS secure!
